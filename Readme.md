
# notificationCall

  ajax call for notification summaries.

## Installation

    $ component install tylergraf/alertserviceCaller

## Example
```js

  var userId = 'cis.something',
  authToken = 'authToken';
  require('notificationCall')(userId, authToken);

```
## API

### require('notificationCall')(userId, authToken);

  Params:
  - userId (string): cis.user.id
  - authToken (string): sessionId
